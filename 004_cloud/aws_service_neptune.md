# Neptune

## Neptune ML

グラフデータに対して予測を行うことができる機能。
GNNの利用により、グラフ構造とプロパティの双方を使用したモデル構築が可能。
またプロパティなどを数値データに反感するために、複数のテキストエンコードを使用することができる。
