# E2Eテストツール

- ブラウザ上で操作するだけで操作を覚えてその後自動で同じテストを行ってくれるツール。

- また、メールのテストも行ってくれます。リンク切れ問題などはこちらで解消できるはずです。

- Mabl
  - https://www.mabl.com/product

- Autify
